# BookWorm Backend API

BookWorm is a personalized **book recommendation & reading tracker backend** built with **Node.js, Express, MongoDB, and TypeScript**.  
It supports **Books, Genres, Reviews, Tutorials, Users, and Personal Reading Shelves**, fully CRUD-ready, and easy to test with Postman.

---

## Tech Stack

- Node.js + Express  
- MongoDB + Mongoose  
- TypeScript  
- bcryptjs for password hashing  
- Optional: JWT for future authentication

---

## Environment Variables

- MONGO_URI=<your_mongodb_connection_string>
- PORT=5000
